> **"Experience visual motion. Explore the unseen."**  
> Pixora is a high-end, immersive gallery designed for the modern web. Built with a focus on motion, aesthetics, and user experience, it serves as a showcase for cinematic video and high-fidelity photography.

---

## ⚡ Key Highlights

- **🎯 Immersive Cinema Player**: A custom-built watch experience on `watch.html` with interactive "Up Next" video linking and smooth transitions.
- **✨ Premium Micro-Interactions**: Features a magnetic button system, custom cursor tracking, and elegant glassmorphic UI elements.
- **🖼️ Masonry Gallery Engine**: A responsive, self-organizing photo grid on `gallery.html` that maintains image aspect ratios perfectly.
- **🎥 Motion-First Design**: Heavily integrated with scroll-reveal animations (`Intersection Observer API`) to make the interface feel alive as you explore.
- **📱 Fully Responsive**: Crafted for every screen size using a blend of modern CSS Grid/Flexbox and minimal Bootstrap 5 utilities.

## 🛠️ Built With

Pixora is a testament to the power of the modern "vanilla" web. It uses zero heavy frameworks for the frontend to ensure maximum performance.

- **Frontend**: HTML5, CSS3, ES6 JavaScript
- **Utilities**: Bootstrap 5 (Grid system only)
- **Icons**: [Favicon.io](https://favicon.io/)
- **Fonts**: [Google Fonts (Inter / Outfit / Roboto)](https://fonts.google.com/)

## 🚀 Quick Setup

Since this is a static project, there are no dependencies to install and no build steps to run.

1. **Clone the project**
   ```bash
   git clone https://github.com/KeshavxGupta/IMAGE-GALLERY.git
   cd IMAGE-GALLERY
   ```

2. **Open in Browser**
   - Simply open `index.html` in your browser.
   - For the best experience (and to avoid local video file handling restrictions), we recommend using **[VS Code Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)**.

## 📁 Project Structure

```text
├── CSS/                  # Compiled styles (global and page-specific)
├── js/                   # Vanilla JavaScript (main.js, Cursor logic, Nav)
├── images/               # Curated photography assets
├── videos/               # High-definition cinematic MP4s
├── index.html            # Main Landing Experience
├── watch.html            # The Cinematic Player
├── gallery.html          # Photography Portfolio
└── explore.html          # Secondary exploration page
```
